Статическая папка `assets`
===========================

Сюда складываются все статические файлы бота:

- `assets/images` — картинки (в том числе фон/обложка главного экрана).
- `assets/guides` — файлы по гайдам.
  - `assets/guides/law161` — материалы по гайду «Гайд: снятие ФЗ-161».
  - `assets/guides/fake_check` — материалы по гайду «Гайд: как определить фейк-чек?».
  - `assets/guides/warmup_2025` — материалы по гайду «Гайд по прогреву карты в 2025 году».

Рекомендуемая структура файлов:

- Картинка на главном экране: `assets/images/main.png`
- Файлы к гайдам:
  - `assets/guides/law161/guide.pdf`
  - `assets/guides/fake_check/guide.png`
  - `assets/guides/warmup_2025/guide.pdf`

После того как положите файлы по указанным путям, бот начнет их отправлять автоматически.
